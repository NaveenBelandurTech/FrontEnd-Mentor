# News Homepage – Frontend Mentor Challenge

This repository contains my solution to the **Frontend Mentor – News Homepage** challenge. The objective was to recreate a responsive news homepage based on provided desktop and mobile JPG design references.

**Live Preview:**  
https://front-end-mentors-news-page.netlify.app/

---

## Overview

### The Challenge

Users should be able to:

- View the optimal layout depending on their device’s screen size.
- See hover and focus states for all interactive elements.

### Project Features

- Responsive navigation (mobile menu and desktop layout)
- Featured article section with hero image
- Sidebar “New” articles block
- Article list with interactive hover states
- Consistent typography and spacing based on the design specification

---

## Technologies Used

- HTML5  
- CSS3 (Flexbox, Grid)  
- JavaScript (basic interactivity)  
- Responsive Web Design  
- Netlify for deployment
